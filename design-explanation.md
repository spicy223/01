# Suanlala’s Portfolio 設計說明

## 一、主題定位

本網站以「水果行」作為個人作品集的核心隱喻，將每件作品視為新鮮上架的當季水果。品牌名稱使用 **Suanlala’s Portfolio**，透過番茄標誌、鮮明色彩與親切文字，傳達酸甜、活潑、好奇且帶有個人溫度的創作者形象。網站主要內容包含作品介紹、關於我、贊助，以及插畫、海報、文字設計與攝影等獨立作品頁。

## 二、設計風格與概念

視覺採用米白、亮黃、橘、粉紅、深綠與深紫色票，搭配黑色粗線、純黑文字陰影、膠囊圓角與幾何水果圖形，呈現簡約可愛的水果攤氛圍。首頁以黃色單色漸層銜接上下段落，作品區使用不同尺寸與角度的流動卡片，滑鼠移動時產生視差，Hover 時淡入作品名稱。關於我頁加入照片翻面、水果掉落與移動籃子；贊助頁則提供金額按鈕與對應 QR Code。

## 三、Block／Flex／Grid 使用說明

### Block

`main`、`section`、作品說明與各內容段落採用 Block 排列，建立由上而下的清楚閱讀順序。每個區塊可獨立設定背景、間距與邊框，也便於響應式調整。

### Flex

Flex 使用於頁首導覽、頁尾、按鈕群組、技能標籤、聯絡區與部分文字配置。`display: flex` 搭配 `align-items`、`justify-content` 與 `flex-wrap`，讓內容能水平對齊，並在空間不足時自然換行。

### Grid

Grid 使用於關於我頁、作品詳情的左右排版，以及海報、文字與攝影作品集。透過 `grid-template-columns` 配置欄寬，再以媒體查詢於平板及手機切換成單欄。首頁流動作品區因需自由排列，另外使用 `position: absolute`、CSS 自訂變數與 JavaScript 指標座標控制視差。

## 四、響應式與互動

網站以媒體查詢支援桌機、平板與手機。文字使用 `clamp()`、`text-wrap: balance` 與 `text-wrap: pretty` 改善閱讀；流動卡片在觸控裝置改為穩定格狀排列。網站亦支援 `prefers-reduced-motion`，降低不適合動態效果使用者的動畫量。
